# ⛳ Golf Charity Platform

A modern full-stack web application where users can **enter golf scores, support charities, and win prizes through weekly draws**.

---

## 🚀 Live Demo

- 🌐 Website: https://golf-charity-app-ten.vercel.app/
- 👤 Dashboard: https://golf-charity-app-ten.vercel.app/dashboard
- 🛠 Admin Panel: https://golf-charity-app-ten.vercel.app/admin 

---

## ✨ Features

### 👤 User Side
- Register & Login
- Enter golf scores (1–45)
- Stores last 5 scores only
- Select a charity to support 💚
- Participate in weekly prize draws
- View draw history & results

---

### 🛠 Admin Panel
- Run draw with random numbers 🎲
- Automatically calculates winners
- Tier-based winning system:
  - 🥇 Tier 5 → 5 matches
  - 🥈 Tier 4 → 4 matches
  - 🥉 Tier 3 → 3 matches
- View all winners
- Leaderboard (top users by wins)
- Platform stats (users, draws, winners)

---

### 🎯 UI/UX Highlights
- Clean, modern dark UI
- Emotion-driven charity-focused design
- Micro-interactions & smooth animations
- Winner highlight effects ✨
- Responsive layout

---

## 🧠 How It Works

1. User enters golf scores  
2. Selects a charity  
3. Admin runs weekly draw  
4. Matching scores → user wins  
5. More matches = higher tier  

---

## 🛠 Tech Stack

- **Type:** Full-Stack Web Application  
- **Frontend:** Next.js (App Router), React, Tailwind CSS  
- **Backend & Database:** Supabase (PostgreSQL, Authentication, Realtime APIs)  
- **Deployment & Hosting:** Vercel  
- **Architecture:** Serverless
